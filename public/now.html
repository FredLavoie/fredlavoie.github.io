What I'm doing now

These past couple of months I've been working on making progress on my structural analysis project. I've spent some time defining the goal of the project, and revising the path I want to take to get there. I've set up the GitHub Projects tab in the repository to outline the major milestones, list the bugs I find and list feature ideas that I have as they come up in my mind. I find GitHub Projects really helpful for keeping the management of the development organized - clearly showing what's been done, what's currently in development and what still needs to be done.

In this last quarter of 2020, I want to make a push to finish reading some textbooks I started earlier this year so I can move on to some new ones in the new year. I bought a behemoth of a textbook recently called TCP/IP Guide and while I don't intend on reading the book in its entirety, I want to dive into some of the sections to better understand the inner workings of the internet, the OSI model and various protocols such as SMTP, FTP and IMAP/POP3.
